        gg.num[gg.s==uus[i]]<-tts[i]

    }
  }

  #cat(rr,"\n")

Here we use the temperature data to identify problem chambers. For thermal tolerance, if the average temperature over the assay is less than 35, we flagged it as problematic. For learning and memory, if the absolute value of the average temperature was greater than 5, we flagged it as problematic. There are very few chambers flagged as problematic for learning that are not also problematic for thermal tolerance (17 records). There are more that are flagged for thermal tolerance but not for learning (7049 records). This could be occuring when the punishment side is NOT the side with the problem.
